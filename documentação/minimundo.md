# Descrição do Minimundo

Este projeto representa um sistema de controle de vendas para uma loja de varejo. O minimundo contempla os seguintes elementos:

- **Clientes**: pessoas que realizam compras na loja.
- **Produtos**: itens disponíveis para venda.
- **Vendas**: transações realizadas entre clientes e a loja.
- **Funcionários**: responsáveis pelo atendimento e registro das vendas.
- **Pagamentos**: formas de pagamento utilizadas nas transações.

O objetivo é modelar um banco de dados relacional que permita registrar e consultar essas informações de forma eficiente e segura.
